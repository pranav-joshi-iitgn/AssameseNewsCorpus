শেহতীয়াকৈ মাইক্ৰ’ছফটে Windows 11মুকলি কৰাৰ কথা ঘোষণা কৰিছে। ইয়াকে লৈ য়ুজাৰৰ মাজত উৎসাহো বৃদ্ধি পাইছে।
কিন্তু য়ুজাৰৰ এনে উৎসাহৰ সুযোগ লৈছে হেকাৰে। তেওঁলোকে ইতিমধ্যে নকল Windows 11 প্ৰস্তুত কৰি উলিয়াইছে।
এই নকল Windows 11 ডাউনলোড কৰি ইনষ্টল কৰিলেই হেকাৰৰ কবলত পৰিব পাৰে য়ুজাৰ।
কম্পিউটাৰৰ বাবে এন্টিভাইৰাছ প্ৰস্তুত কৰা সংস্থা কাছপাৰস্কি (Kaspersky)ৰ প্ৰতিবেদনত এই কথা প্ৰকাশ পাইছে।কাছপাৰস্কিৰ প্ৰতিবেদন অনুসৰি নকল Windows 11 ইনষ্টল কৰাৰ হাৰ পূৰ্বতকৈ বহু বৃদ্ধি পাইছে।
আনহাতে, হেকাৰসকলে য়ুজাৰসকললৈ Windows 11 ৰ ইনষ্টল কৰাৰ লিংক সৰবৰাহ কৰিছে। এনে  Windows 11 কোনোৱে ইনষ্টল কৰিলেই য়ুজাৰৰ কম্পিউটাৰটো হেকাৰৰ নিয়ন্ত্ৰণলৈ গুচি গৈছে।
মাইক্ৰ’ছফটৰ Windows 11 ডাউনলোড তথা ইনষ্টলৰ প্ৰক্ৰিয়া য়ুজাৰৰ বাবে অত্যন্ত সহজ কৰি দিয়া হৈছে। তথাপি বহুতেই আন স্থানৰ পৰা Windows 11ডাউনলোড বা ইনষ্টল কৰিছে।ফলত হেকাৰসকলৰ কাম সহজ হৈ পৰিছে।
আনহাতে, বহুসময়ত অফাৰৰ নামতো য়ুজাৰক প্ৰৱঞ্চনা কৰা হৈছে। কিন্তু কেনেকৈ বাচিব ইয়াৰ পৰা?
প্ৰতিবেদনত য়ুজাৰসকলক স্পষ্ট কৰি জনাই দিয়া হৈছে যে এনে ধৰণৰ লিংক ক্লিক কৰি Windows 11 ডাউনলোড বা ইনষ্টল কৰিলেই হেকাৰৰ কবলত পৰিব লাগিব। কেবল মাইক্ৰ’ছফটৰ ৱেবছাইটৰ পৰাহে য়ুজাৰক Windows 11 ডাউনলোড তথা ইনষ্টল কৰাৰ পৰামৰ্শ দিয়া হৈছে।
