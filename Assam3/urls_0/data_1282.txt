কাইলৈৰ পৰা ২৩ মাৰ্চলৈ তামুলপুৰত অনুষ্ঠিত হ’ব লগা চতুৰ্থ এছিয়ান খো খো চেম্পিয়নশ্বিপ প্ৰতিযোগিতাৰ বাবে আজি ভাৰতীয় পুৰুষ আৰু মহিলা দল দুটা ঘোষণা কৰা হয়৷ আজি সন্ধিয়া তামুলপুৰত এক সংবাদ মেল অনুষ্ঠিত কৰি ভাৰতীয় দল দুটা ঘোষণা কৰে ভাৰতীয় খো খো ফেডাৰেছনৰ অধ্যক্ষ সুধাংশু মিট্টালে।

ঘোষিত পুৰুষ দলটো আফছে ভাংগাৰে ( দলপতি , মহাৰাষ্ট্ৰ ) , আফছে গানপাটে ( উপ দলপতি , ভাৰতীয় ৰেল বিভাগ ) , অমিত পাটিল ( ভাৰতীয় ৰেল বিভাগ ) , অৰুণ গুণকী ( ৰেল বিভাগ ) , সূয়াছ গাৰ্গে ( মহাৰাষ্ট্ৰ ) , অংকিত পুতে ( মহাৰাষ্ট্ৰ ) , গৌথাম এম কে ( কৰ্ণাটক ) , আতলা শিৱ নাগাৰেড্ডী ( অন্ধ্ৰ প্ৰদেশ ) , মদন ( দিল্লী ) , এছ সন্ত হুৰু ( তামিলনাডু ) , ভি কাবিলন ( পণ্ডিচেৰী ) , শচীন ভাৰগু ( মধ্য ভাৰত ) , আকাশ ( উত্তৰ প্ৰদেশ ) , মুকেশ ( ৰাজস্থান ) । ষ্টেণ্ড বাই — শুভম কাঁচি ( মধ্য প্ৰদেশ ) , শিবিন এম ( কেৰেলা ) , ধ্ৰুৱ ( হাৰিয়ানা ) , তীৰ্থ ডেকা ( অসম )।
আনহাতে মহিলা দলটো নাশ্ৰীন ( দলপতি , এ এ আই ), পাৰবিন নিচা ( উপ দলপতি, দিল্লী ) , প্ৰিয়ংকা ( মহাৰাষ্ট্ৰ ), গৌৰী চিন্ধে ( মহাৰাষ্ট্ৰ ), অপেক্ষা সূতাৰ ( মহাৰাষ্ট্ৰ ), নিকিতা পাৱাৰ ( এ এ আই ), মনিকা এল ( কৰ্ণাটক ), বিন্দু ( হাৰিয়ানা ), নিৰ্মলা ভাটি ( ৰাজস্থান ), গুৰবীৰ কৌৰ ( পাঞ্জাৱ ) , অৰ্চনা মাজি ( উৰিষ্যা ), দীপিকা চৌধুৰী ( পশ্চিম বংগ ), মিনু ( হাৰিয়ানা), ৰঞ্জনা শৰণীয়া ( অসম ), মধু ( দিল্লী )। ষ্টেণ্ড বাই — ঋত্বিকা ( মধ্য বিহাৰ ), মণিকা ( বিহাৰ ), ৰেচমা ( মহাৰাষ্ট্ৰ )।

কাইলৈ ভুটান ,মালয়েছিয়া ,ইণ্ডোনেচিয়া আৰু দক্ষিণ কোৰিয়াৰ মাজত উদ্বোধনী খেল অনুষ্ঠিত হ’ব। দিনৰ এক বজাত অনুষ্ঠিত হবলগীয়া উদ্বোধনী অনুষ্ঠানত অংশ গ্ৰহণ কৰিব উত্তৰ পূৱ ভাৰতৰ জি অ চি লেফটেনেন্ট জেনেৰেল ৰাণা প্ৰতাপ কলিতা, অসমৰ প্ৰাক্তন মুখ্য সচিব সঞ্জয় কৃষ্ণ,বিটিআৰ ৰ প্ৰধান প্ৰমোদ বড়োই।
